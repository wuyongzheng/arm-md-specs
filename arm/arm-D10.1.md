## D10.1 Introduction

IQGVTK

The Memory Tagging Extension (MTE) describes the following concepts:

- Allocation Tags that are held at Tag locations and are associated with blocks of memory called Tag Granules .
- Logical Address Tags that form the upper bits of addresses used for Tag Checked memory accesses.
- Tag Check operations in which the Allocation Tags and Logical Address Tags are compared.
- Tag Check Faults that are generated from unsuccessful Tag Check operations .

For more information, see:

- Allocation Tags.
- Tag Checked memory accesses.
- Tag Check operations.
- Tag Check Faults.
